# Chess Opening Intelligence

> **Data Engineering + Data Analytics Portfolio Project**

Identifying the "best ROI" chess openings for specific Elo ratings using the **Opening Intelligence Index (OII)**.

---

### 📊 The Metric: Opening Intelligence Index (OII)
Most chess analytics show win rates, which are biased by Elo differences. The OII adjusts performance for:
1. **Elo Difference:** Subtracts the "expected score" from actual results.
2. **Tactical Complexity:** Weights performance against the difficulty of puzzles generated by that opening.
3. **Frequency:** Normalizes for sample size.

**Example:** The London System might have a high win rate at 1200 Elo, but if those wins are against 1000-rated players, its OII will be low.

### 🛠️ Tech Stack
- **Python:** PGN parsing and pipeline orchestration.
- **DuckDB:** Local analytical SQL engine for fast Parquet processing.
- **SQL:** CTEs and Window Functions for all metric calculations.
- **Power BI:** Interactive dashboard for opening ROI visualization.

### 🚀 How to Run (MVP)
1. **Clone the repo**
2. **Install dependencies:** `pip install -e .`
3. **Run the puzzle pipeline:** `python pipeline/02_ingest_puzzles.py`
4. **Run the games pipeline:** `python pipeline/01_ingest_games.py`
